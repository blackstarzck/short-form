# 페이지별 상세 설계서 (Final)

## ① [메인] 지식 발견 피드 (Reels Style)
* **UI 레이아웃:**
    * **사이드 바 (최소화):** 프로필, 북마크(저장), 더보기(`...`) 버튼.
    * **자막 박스:** 하단 시스템 영역 60px 위 배치 (고대비 반투명 박스).
    * **인사이트 버튼:** 자막 상단 플로팅 버튼 (심화 학습 진입).
* **더보기(...) 액션 시트:** 좋아요, 댓글, 공유하기, 신고, **도서 구매 링크** 통합.
* **엔딩 카드 (Ending Card):** 영상 종료 시 '다음 클립 보기', '책 상세 보기' 버튼 노출.

## ② [심화] 지식 상세 및 액션 시트 (Deep Dive)
* **콘텐츠:** Gemini가 추출한 요약 카드뉴스, 핵심 키워드, 관련 도서 정보.
* **비즈니스 전환:** '전체 내용 보기' 클릭 시 전자책 결제 또는 구독 페이지로 유도.

## ③ [성장] 나의 지식 대시보드 (Dashboard)
* **시각화:** 카테고리별 습득 지식을 성좌(Recharts) 형태로 표현.
* **인증 센터:** 마이크로 러닝 코스 완료 시 발급된 '디지털 수료증' 목록 관리.
