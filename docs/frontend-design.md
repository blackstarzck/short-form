# 프론트엔드 디자인 가이드라인 (Final)

## 1. 지식 전달 최적화 (Visual Rules)
* **3초 비주얼 훅:** 영상 시작 시 중앙에 핵심 질문이나 강렬한 타이포그래피 노출.
* **자막 가독성:** 배경색에 관계없이 잘 보이도록 블랙 반투명 박스(Opacity 0.6) 필수 적용.
* **유형별 프리셋:** '북 썸머리'는 텍스트 강조 레이아웃, '전문가 영상'은 화자 강조 레이아웃 적용.

## 2. 행동 유도 및 보상 (Interactions)
* **CTA 버튼:** 구매 전환 버튼은 브랜드 컬러(#FF9800)와 높은 대비를 사용하여 시선 유도.
* **학습 피드백:** 북마크 시 성좌 그래프에 즉시 별이 추가되는 애니메이션 효과.
* **딥 서치:** 롱프레스 시 해당 타임라인의 자막 텍스트를 즉시 복사/공유.

## 3. 접근성
* 모든 버튼은 최소 44x44px 터치 영역 확보.
* 소리 없이 자막만으로도 핵심 지식의 70%를 이해할 수 있도록 설계.
