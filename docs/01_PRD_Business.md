# 📖 비즈니스 요구사항 문서 (PRD)

## 1. 프로젝트 비전
* **서비스명:** KEDUALL (지식 분절화 플랫폼)
* **핵심 가치:** "방대한 도서 지식을 60초의 강렬한 경험으로 분절(Fragmentation)하여 전달한다."

## 2. 핵심 비즈니스 모델 (BM)
1. **Micro-Learning:** 도서 1권을 10~15개의 쇼츠 클립으로 쪼개어 유기적인 학습 코스로 구성.
2. **Hooking & Funnel:** 쇼츠로 호기심을 유도한 뒤, 전체 내용을 위한 전자책 결제/구독으로 전환.
3. **Watch-to-Earn (W2E):** 시청 완료 시 학습 포인트를 지급하여 지속적인 학습 동기 부여.
4. **Seamless Reading:** 쇼츠 시청 중 즉시 도서 본문을 열람할 수 있는 '바로 읽기' 경험 제공.

## 3. 핵심 정책 (Policy)
* **포인트 지급:** 영상의 90% 이상 시청 시 포인트 적립 (중복 시청 제외).
* **콘텐츠 구조:** 도서 데이터와 연동된 쇼츠 클립 관리 및 시리즈화.
* **바로 읽기 접근성:** 사용자가 영상 시청 중 언제든 도서 본문으로 진입할 수 있도록 숏폼 UI 내 전용 진입점(Book Reader) 배치.
